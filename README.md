# Calculator

Simple HTML/CSS/JS calculator. Open `index.html` in a browser to use.

Keyboard shortcuts:
- Digits `0-9` and `.` to enter numbers
- `+ - * /` for operators
- `Enter` or `=` to evaluate
- `Backspace` to delete last character
- `Escape` to clear (AC)
