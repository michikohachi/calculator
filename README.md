# Calculator

Simple HTML/CSS/JS calculator. Open `index.html` in a browser to use.

Keyboard shortcuts:
- Digits `0-9` and `.` to enter numbers
- `+ - * /` for operators
- `Enter` or `=` to evaluate
- `Backspace` to delete last character
- `Escape` to clear (AC)

Files:
- `index.html` — UI
- `style.css` — styles
- `script.js` — logic and keyboard support

To open locally (Windows):

1. In Explorer, double-click `index.html`.
2. Or from PowerShell:

```powershell
start "" "C:\Users\Leizel\OneDrive\Desktop\Files\carl\finalsemProject\calculator\index.html"
```
